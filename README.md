# Wetland Mapping - ELEC 498 Capstone

Automated wetland classification using Google Earth Engine satellite embeddings and machine learning.

**Team:** Jacob Badali, Connor Rewa, Max Marsland, Gavin Nyhof (Group 46)

## Dataset

- **Region:** Bow River Basin, Alberta, Canada
- **Classes:** 6 wetland types (0-5) including background
- **Samples:** 1.5M balanced training samples
- **Features:** 64-dimensional Google Earth embeddings
- **Resolution:** 10m spatial resolution

## Key Files

- **Training Data:** `wetland_dataset_1.5M_4Training.npz` (hosted on Google Drive)
- **Labels:** `bow_river_wetlands_10m_final.tif`
